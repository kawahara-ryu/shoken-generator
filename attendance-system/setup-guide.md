## 2. スプレッドシートの「下地」作成手順

### Step 1: シートを作成

1. Googleスプレッドシートを新規作成
2. シート名を以下に変更:
   - シート1 → **メイン**
   - シート追加 → **ログ**
   - シート追加 → **分析**

### Step 2: 自動セットアップの実行

1. メニュー「拡張機能 > Apps Script」を開く
2. `Code.gs` の全コードを貼り付けて保存
3. 関数選択で `setupSpreadsheet` を選び ▶ 実行
4. 初回は権限承認ダイアログが出るので「許可」

> これで背景色・フォント色・ヘッダー・列幅・条件付き書式が一括適用されます。

### Step 3: 固定行の設定

1. メインシートを開く
2. 行2を選択
3. メニュー「表示 > 固定 > 2行まで」

### Step 4: 図形ボタンの配置

1. メニュー「挿入 > 図形描画」
2. 角丸四角形を描き、テキスト「1組」を入力
3. スタイル設定:
   - 背景色: `#1a1a2e`（ダークネイビー）
   - 枠線色: `#00f2ff`（ネオンブルー）
   - 枠線太さ: 2px
   - テキスト色: `#00f2ff`
   - フォントサイズ: 10pt / 太字
4. 「保存して閉じる」 → 行1〜2の領域に配置
5. 同様に「2組」〜「6組」「分析」ボタンを作成

### Step 5: 名簿の入力

- B列に各クラスの生徒氏名を入力（またはコピー&ペースト）
- A列の出席番号は `setupSpreadsheet` で自動入力済み

## 3. 条件付き書式レシピ

> `setupSpreadsheet()` を実行すると自動で適用されますが、手動で設定する場合は以下を参照。

### 適用範囲

各クラスのデータ行全体（例: 1組は `A5:G46`）

| 条件（D列の値） | 行の背景色 | 意味 |
|:---:|:---:|:---:|
| `欠席` | `#330000`（ダークレッド） | 欠席者を赤く強調 |
| `遅刻` | `#333300`（ダークイエロー） | 遅刻者を黄色く強調 |
| `早退` | `#003333`（ダークシアン） | 早退者をシアンで強調 |
| `保健室` | `#331a00`（ダークオレンジ） | 保健室移動をオレンジ点灯 |
| `出停` | `#1a001a`（ダークマゼンタ） | 出席停止をマゼンタ点灯 |

### 手動設定手順

1. 対象範囲を選択（例: `A5:G46`）
2. メニュー「表示形式 > 条件付き書式」
3. 「カスタム数式」を選択
4. 数式: `=$D5="欠席"` → 背景色 `#330000` に設定
5. 同様に他のステータスも追加

## 4. ボタンにスクリプトを割り当てる手順

### 基本手順

1. メインシート上の図形ボタン（例:「1組」）を **右クリック**
2. メニューから **「スクリプトを割り当て」** を選択
3. 関数名を入力（`jumpToClass1` など）→ OK

### 割り当て表

| ボタン | 関数名 |
|:---:|:---:|
| 1組 | `jumpToClass1` |
| 2組 | `jumpToClass2` |
| 3組 | `jumpToClass3` |
| 4組 | `jumpToClass4` |
| 5組 | `jumpToClass5` |
| 6組 | `jumpToClass6` |
| 分析 | `jumpToAnalysis` |
| 📊 ダッシュボード更新 | `refreshHologramCharts` |

### トリガーの設定

`Code.gs` の `createDailyTriggers()` を一度だけ手動実行してください。以下のトリガーが自動登録されます:

| 時刻 | 関数 | 動作 |
|:---:|:---:|:---:|
| 毎日 06:00 | `dailyReset` | メインシートの理由・備考欄をクリア |
| 毎日 18:00 | `dailyArchive` | 本日の全データをログシートに転記 |
